Featured web white board with pure html, css & js.  
Along with screen sharing feature.

<h3> Features: </h3>

[x] Undo & Redo  
[x] Clearing & Deleting boards  
[x] Downloading boards  
[x] Multiple boards  
[x] Zoom in or out board  
[x] Multi colored & sized pen & Eraser  
[x] Multiple shapes
